Inception Project - Static Website

Author: hheng

About

This is a static website created as a bonus service for the Inception project. It demonstrates setting up additional Docker containers beyond the mandatory requirements.

Services

Mandatory:

  • ✓ NGINX (Web Server with TLS)
  • ✓ WordPress (PHP-FPM)
  • ✓ MariaDB (Database)

Bonus:

  • ✓ Redis (Cache)
  • ✓ FTP Server
  • ✓ Adminer (Database Management)
  • ✓ Portainer (Container Management)
  • ✓ Static Website (This page)

Technologies

Docker • Docker Compose • NGINX • MariaDB • WordPress • PHP-FPM • Redis • vsftpd
